## Deployment

To deploy this project run

```bash
  yarn run deploy
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/livingdolls/typescript-pos
```

Go to the project directory

```bash
  cd typescript-pos
```

Install dependencies

```bash
 yarn  install
```

Start the server

```bash
  yarn ts
```

```bash
  yarn dev
```
